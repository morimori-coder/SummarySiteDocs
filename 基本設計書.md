# 基本設計書

## 概要
社交ダンス教室のまとめサイトを作成する。  
主な機能は以下。  
+ 教室情報検索機能
+ 直近の検索条件保持機能(できれば)
+ 直近の検索結果保持機能(できれば)
+ React Routerを利用したSPA
